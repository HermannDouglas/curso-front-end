# Unidade II - Introdução à JavaScript

O que é?

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos, utilizada principalmente para adicionar interatividade a páginas web

A importância do JavaSript no desenvolvimento web
  • Interatividade do lado do cliente: O JavaScript permite adicionar interatividade e dinamismo às páginas da web. Com ele, é possível criar elementos interativos, como botões clicáveis, menus expansíveis, formulários validados, carrosséis de imagens e muito mais.
  • Client-side scripting: JavaScript é executado no lado do cliente, ou seja, no navegador do usuário. Isso permite que você manipule elementos HTML, responda a eventos e interaja com o usuário diretamente na página, sem a necessidade de recarregar a página.
  • Sintaxe: A sintaxe do JavaScript é semelhante a outras linguagens de programação, como C, C++ e Java. Ela inclui regras para a estrutura dos comandos, declaração de variáveis, definição de funções, entre outros.
  • Variáveis: Em JavaScript, você pode declarar variáveis usando as palavras-chave var, let ou const. Variáveis são usadas para armazenar valores, como números, strings, objetos e outros tipos de dados.
  • Tipagem dinâmica: JavaScript é uma linguagem de tipagem dinâmica, o que significa que você não precisa declarar explicitamente o tipo de uma variável. O tipo de uma variável pode mudar durante a execução do programa.
  • Funções: Funções são blocos de código que podem ser reutilizados em diferentes partes do programa. Elas podem receber parâmetros (valores de entrada), executar uma série de instruções e retornar um valor.
  • Manipulação do DOM: O Document Object Model (DOM) é uma representação em árvore dos elementos HTML de uma página. Com JavaScript, você pode acessar e manipular o DOM para adicionar, remover ou alterar elementos HTML, estilos, conteúdo e eventos.
  • Eventos: JavaScript permite que você responda a eventos, como cliques de mouse, envio de formulários, carregamento da página, entre outros. Você pode atribuir funções a eventos específicos para executar ações quando o evento ocorrer.
  • Condicionais e Loops: JavaScript oferece estruturas de controle, como , else if, else e switch, para executar diferentes blocos de código com base em condições. Além disso, você pode usar loops, como for e while, para repetir a execução de um bloco de código várias vezes.
  • Bibliotecas e frameworks: Existem várias bibliotecas e frameworks populares em JavaScript, como jQuery, React, Angular e Vue.js. Essas ferramentas fornecem conjuntos de funcionalidades pré-construídas para facilitar o desenvolvimento de aplicações web mais complexas.
